## 🎬 交互动效

### 页面切换
```css
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
```
- 进入：从下方淡入 + 上移 20px
- 离开：向上淡出 + 上移 -20px

### 卡片悬停
```css
transform: translateY(-2px);
box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1);
```

### 按钮交互
```
默认 → 悬停 → 点击
100ms ease 过渡
```

## 📱 响应式设计

### 桌面 (>1024px)
- 侧边栏固定 240px (可折叠 64px)
- 统计卡 4 列
- 图表 2 列

### 平板 (640-1024px)
- 侧边栏抽屉式
- 统计卡 2 列
- 图表 2 列

### 手机 (<640px)
- 侧边栏隐藏
- 统计卡 1 列
- 图表 1 列
