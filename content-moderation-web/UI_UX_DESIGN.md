## 🎨 设计理念

### 设计原则
1. **专业可靠** - 企业级 B 端产品设计语言
2. **清晰高效** - 信息层次分明，操作便捷
3. **现代美观** - 符合 2026 年设计趋势
4. **一致规范** - 统一的视觉语言和交互规范

## 🎨 色彩系统

```

### 中性色
```

## 📐 布局系统

### 响应式断点
```
Mobile:  < 640px
Tablet:  640px - 1024px
Desktop: > 1024px
Wide:    > 1440px
```

### 间距规范
```
4px 体系：4, 8, 12, 16, 20, 24, 32, 40, 48, 64
```

### 圆角规范
```
Small:  4px   (按钮、标签)
Medium: 8px   (卡片、输入框)
Large:  12px  (弹窗、抽屉)
Round:  9999px (头像、徽章)
```

## 🎬 交互动效

### 过渡动画
```
页面切换：300ms cubic-bezier(0.4, 0, 0.2, 1)
卡片悬停：200ms ease
按钮点击：100ms ease
数据加载：骨架屏 + 淡入
```

### 加载状态
```
- 列表加载：骨架屏
- 按钮加载：旋转图标 + 禁用
- 图表加载：进度条 + 占位图
```

### 空状态
```
- 图标：64x64 中性色图标
- 标题：16px 中粗体
- 描述：14px 中性色 500
- 操作按钮：主要按钮
```

## 📱 响应式设计

### 移动端适配
```
侧边栏：抽屉式，可滑动
统计卡：单列显示
图表：100% 宽度，高度自适应
表格：横向滚动或卡片式展示
```

### 平板适配
```
侧边栏：固定 240px
统计卡：2 列显示
图表：2 列布局

### 桌面适配

侧边栏：固定 240px (可折叠 64px)
统计卡：4 列显示
图表：2 列布局
表格：正常显示
```

## 🎯 用户体验优化

### 1. 快捷操作
- 全局搜索：Ctrl+K 唤起
- 常用功能：添加到收藏夹
- 批量操作：复选框 + 底部操作栏

### 2. 数据展示
- 数字格式化：千分位分隔
- 时间格式化：相对时间 + 绝对时间
- 状态展示：彩色标签 + 图标

### 3. 错误处理
- 表单验证：实时验证 + 提交验证
- 错误提示：Toast + 行内错误
- 异常页面：404/500 自定义页面

### 4. 性能优化
- 虚拟滚动：长列表优化
- 图表懒加载：可见区域渲染
- 数据缓存：SWR 模式
