# 用户画像系统 - 前端设计文档

## 🎯 系统定位

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   核心问题：这个人是怎么样的人？                             │
│                                                             │
│   输入：视频内容 → AI 分析 → 违规识别                        │
│   输出：用户画像 → 供业务系统使用                            │
│                                                             │
│   ❌ 不是：内容审核管理平台                                  │
│   ✅ 是：用户画像数据服务                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## 📊 画像维度设计

### 用户画像金字塔

```
                    ┌─────────────┐
                    │  综合画像   │
                    │ 风险等级    │
                    │ 信任分      │
                    └──────┬──────┘
                           │
          ┌────────────────┼────────────────┐
          │                │                │
    ┌─────┴─────┐    ┌─────┴─────┐    ┌─────┴─────┐
    │ 违规画像  │    │ 行为画像  │    │ 内容画像  │
    │          │    │          │    │          │
    │ 类型分布  │    │ 通话习惯  │    │ 环境评分  │
    │ 违规频率  │    │ 活跃时段  │    │ 互动评分  │
    │ 违规趋势  │    │ 违规率    │    │ 专业度    │
    └──────────┘    └──────────┘    └──────────┘
```

### 画像维度详解

| 维度 | 指标 | 数据来源 | 说明 |
|------|------|----------|------|
| **风险画像** | 风险等级 | 综合计算 | LOW/MEDIUM/HIGH |
| | 信任分 | 综合计算 | 0-100 分 |
| | 风险趋势 | 历史对比 | 改善/稳定/恶化 |
| **违规画像** | 违规类型分布 | 违规事件聚合 | 各类型占比 |
| | 违规频率 | 违规事件统计 | 次/周、次/月 |
| | 主要违规类型 | 违规事件分析 | 最常出现的类型 |
| | 平均置信度 | 违规事件计算 | AI 判定置信度 |
| **行为画像** | 总通话数 | 视频分析任务 | 被分析的通话数 |
| | 违规率 | 违规数/总通话数 | 百分比 |
| | 平均违规时长 | 违规事件计算 | 秒 |
| **内容画像** | 环境评分 | 环境类违规计算 | 0-100 |
| | 互动评分 | 互动类违规计算 | 0-100 |
| | 专业度评分 | 综合计算 | 0-100 |

## 🔌 Mock API 设计

### API 基础信息

```
Base URL: /api/v1
响应格式: JSON
```

### 统一响应结构

```typescript
interface ApiResponse<T> {
  code: number;        // 200 成功，其他失败
  message: string;     // 响应消息
  data: T;             // 响应数据
}
```

### 1. 数据概览 API

#### GET /api/v1/dashboard/statistics

**描述**：获取概览统计数据

**响应**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "totalUsers": 12345,
    "highRiskUsers": 234,
    "mediumRiskUsers": 1567,
    "lowRiskUsers": 10544,
    "trends": {
      "totalUsers": 5.2,
      "highRiskUsers": -2.1,
      "mediumRiskUsers": 0.3,
      "lowRiskUsers": 6.8
    }
  }
}
```

#### GET /api/v1/dashboard/risk-distribution

**描述**：获取风险等级分布

**响应**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "high": 234,
    "medium": 1567,
    "low": 10544
  }
}
```

#### GET /api/v1/dashboard/violation-distribution

**描述**：获取违规类型分布

**响应**：
```json
{
  "code": 200,
  "message": "success",
  "data": [
    { "type": "CALL_IN_BED", "name": "躺在床上通话", "count": 1234 },

    { "type": "SLEEPING", "name": "睡觉", "count": 156 }
  ]
}
```

#### GET /api/v1/dashboard/risk-trend

**描述**：获取风险趋势数据

**参数**：
- `days`: 天数，默认 30
